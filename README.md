# Java Tracing Problems - Conditional Statements & Loops

A comprehensive collection of basic Java programming problems solved with **tracing tables**. This repository focuses on mastering **conditional statements** (`if-else`, `switch`) and **loops** (`for`, `while`, `do-while`) through systematic tracing.

## 📌 About This Repository

This repo is designed for beginners who want to:
- Understand the flow of execution in Java programs
- Master **tracing tables** (manual dry run technique)
- Build strong logic building skills
- Learn how to debug programs step by step

Perfect for students learning Java fundamentals, competitive programming beginners, or anyone preparing for programming interviews.

## 🎯 What You'll Find

- **Tracing Tables** for every problem
- Clear problem statements
- Step-by-step manual tracing
- Well-commented Java code
- Input/Output examples
